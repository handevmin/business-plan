1. 맞춤형 스마트 학습 공간

맞춤형 학습 보조 솔루션 고도화

개발 맞춤형 플래너 제작

개인별 학습 플래너 자동 생성

  • 학습자의 목표, 과목별 수준, 시험 일정 등을 반영한 맞춤형 학습 계획 제공
  • 학습 일정, 과목별 배분 시간, 심화/기본 학습 비율 자동 설계
  • 개인별 학습 패턴과 목표 달성률에 따른 플래너 최적화
  • 시험 일정에 맞춘 역산 학습 계획 자동 조정

학습 로그 연동 및 성과 분석

  • 온라인 학습 플랫폼, 문제 풀이 결과와 자동 연동
  • 주 단위, 월 단위 학습 성과 분석 리포트 제공
  • 취약 영역 자동 감지 및 집중 학습 시간 할당
  • 목표 달성률에 따른 플래너 자동 재설계

플래너를 인공지능 스피커와 연동

  • 개인 학습 플래너 주요 알림(과목별 시작 시간, 휴식 타이밍 등)을 음성으로 안내
  • 학습 진행 상황(공부 시작/종료 시점, 집중 시간 등)을 음성 명령으로 실시간 기록
  • 앱과 자동 연동되어 실시간 학습 활동 추적 및 기록
  • 음성 인식을 통한 간편한 일정 변경 및 조정 기능

실력 부족 항목 분석 및 플래너 보완

그림 2: AI 기반 과목별 성취도 분석 및 부족 항목 시각화

데이터 기반 분석

  • 정답률, 오답 패턴 등을 분석해 우선 보충 과목 또는 문항 유형 식별
  • 반복적인 오답 패턴 발견 및 집중 학습 영역 추천
  • 학습자의 취약점을 자동 분류하여 유형별 대응책 제시
  • 학습 시간 대비 효율성 분석으로 학습법 개선 방향 제안

플래너 자동 조정

  • 다음 주차/월간 플래너에 취약 항목 자동 반영
  • 학습 취약 분야에 추가 시간 배정 및 보충 자료 추천
  • 개인화된 복습 주기 설정으로 망각 곡선 최적화
  • 주도적 학습 관리가 가능하도록 상세 가이드 제공

2. AI 생활 관리 선생님

실시간 동작 감지 모니터링 고도화

학습 공간에 AI 기반 동작 감지 솔루션 적용

그림 3: 학습 공간 내 AI 센서 배치 및 구성도
  • 카메라 센서: 얼굴 표정, 시선, 자세 등을 감지하여 집중도 추적
  • 압력 센서: 책상 의자에 설치하여 착석 여부 및 자세 변화 감지
  • 시선 추적 센서: 학습 자료에 대한 집중도와 읽기 패턴 분석
  • 학습자가 자리를 비우거나 부적절한 자세 유지 시 실시간 경고 발생
  • 스마트 스피커를 통한 음성 피드백 및 알림 제공

카메라를 통한 학습자 집중도 추적

그림 4: AI 기반 학습자 집중도 추적 및 휴식 개입 효과

집중도 지수 산출

  • 얼굴, 시선, 자세 인식으로 집중도 지수 산출
  • 시선 이탈 빈도, 스마트폰 사용, 졸음 여부 감지
  • 장시간 집중 후 피로도 증가 패턴 감지
  • 학습자별 평균 집중 지속 시간 분석

자동 개입 시스템

  • 집중도가 급격히 낮아지면 휴식 안내
  • AI 스피커 또는 스마트 디스플레이로 알림 제공
  • 최적 휴식 시간 및 활동 추천
  • 개인별 학습-휴식 패턴 최적화

  • AI 모델이 피드백 효과를 학습하여 개인화된 개입 전략 발전

집중도 기록 및 차후 플래너 반영

데이터 축적 및 분석

  • 일·주간 단위로 집중도 데이터 축적
  • 개인별 학습 리포트 자동 생성
  • 시간대별, 과목별, 환경별 집중도 패턴 분석
  • 학습 성과와 집중도 상관관계 도출

맞춤형 플래너 연동

  • 개인별 '최적 학습 시간대' 자동 반영
  • '적절한 휴식 주기' 데이터 기반 최적화
  • 집중도가 높은 과목을 어려운 시간대에 배치
  • 학습자의 생체리듬에 맞춘 일정 조정

직관적 질의응답

  • 음성/텍스트로 질문하면 NLP 기반으로 즉시 해설 제공
  • 풀이 과정 단계별 설명 및 개념 이해 지원
  • 연관 개념 및 배경 지식 자동 제공
  • 질문 의도에 맞는 정확한 답변 생성

맞춤형 학습 경로

  • 틀린 문제와 반복 실수 유형에 대한 추가 학습 제안
  • "이 개념 관련 기초 문제도 풀어보세요." 등 추가 학습 경로 제공
  • 학습자 수준에 맞는 단계적 난이도 조정
  • 취약점 극복을 위한 개인별 문제 세트 생성

학습자의 질의 사항, 오답 패턴 등 빅데이터 추적

그림 7: AI 튜터의 학습자 데이터 분석 개념도
  • 자주 묻는 질문, 특정 개념에서의 오답률 등을 AI가 스스로 학습
  • 튜터봇 정확도 지속적 향상 및 답변 품질 개선
  • 이용자의 실력 향상도를 측정해 맞춤형 보충강의 자동 연결
  • 영상 강의, PDF 자료 등 다양한 형태의 학습 자료 제안
  • 개인별 학습 진도와 성취도에 맞춘 콘텐츠 큐레이션
